# Steering Docs (Weaszel)

These docs are meant to help you **onboard fast**, understand the system end-to-end, and see the **highest‑leverage improvements** (speed, reliability, agent intelligence, and maintainability).

### Where to start

- **If you want to run it**: read `STEERING-SETUP.md`
- **If you want to understand the architecture**: read `STEERING-ARCHITECTURE.md`
- **If you want speed + reliability wins**: read `STEERING-PERFORMANCE.md`
- **If you want better prompting / agent behavior**: read `STEERING-PROMPTING.md`
- **If you want “what’s brittle / resisting change”**: read `STEERING-RESISTANCE.md`
- **If you want a map of the repo**: read `STEERING-CODEMAP.md`

### Scope of the repo

- **CLI Agent (the product)**: `job-weasel-agent/` (Python)
- **Website (marketing/docs)**: `web/` (Next.js)
